MC Height Checker 118

This tool allows you to get the terrain height in Minecraft 1.18+. You will need a wasm file to use this.
*It does not use cave noise or ravines at all, so it is not suitable for finding low elevations. However, the heights of terrain affected by windswept features are accurate.

Coordinate Modes:
• Single Point: Check one specific coordinate
• Range: Check all coordinates within X/Z min/max bounds
• Center + Radius: Check coordinates within radius of center point
